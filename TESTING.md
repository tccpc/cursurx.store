# 功能测试清单

使用本文档验证 Cursor X 网站的所有功能是否正常工作。

## ✅ 测试前准备

1. 启动本地服务器：
```bash
npm start
# 或
python -m http.server 8000
```

2. 在浏览器中打开：`http://localhost:8000`

3. 打开浏览器开发者工具（F12）查看控制台，确保没有错误

## 🌍 多语言切换测试

### 测试步骤

1. **初始语言检查**
   - [ ] 页面默认显示英文（如果是首次访问）
   - [ ] 导航栏显示：Pricing, Features, Models, FAQ
   - [ ] Hero 标题显示英文

2. **切换到中文**
   - [ ] 点击右上角的"繁中"按钮
   - [ ] 按钮变为激活状态（高亮）
   - [ ] 所有文本立即切换为繁体中文
   - [ ] 导航栏显示：價格方案, 功能特色, 模型支援, 常見問題

3. **验证中文内容**
   - [ ] Hero 标题：在 Cursor 中無限使用 Claude & GPT...
   - [ ] 促销横幅显示中文
   - [ ] 定价方案标题为中文
   - [ ] 推荐标签显示"推薦"
   - [ ] 购买按钮显示"立即購買"
   - [ ] FAQ 标题和内容为中文

4. **切换回英文**
   - [ ] 点击"EN"按钮
   - [ ] 所有内容切换回英文
   - [ ] 推荐标签显示"Recommended"
   - [ ] 购买按钮显示"Buy"

5. **语言持久化**
   - [ ] 选择中文后刷新页面
   - [ ] 页面仍然显示中文（语言偏好已保存）
   - [ ] 打开开发者工具 > Application > Local Storage
   - [ ] 查看 `language` 键的值为 `zh`

## 🐛 发现问题？

如果发现任何问题：

1. 检查浏览器控制台的错误信息
2. 确认所有文件都正确加载（特别是 `i18n.js`）
3. 清除浏览器缓存后重试
4. 检查 `config.js` 和 `i18n.js` 的语法
5. 查看 `LANGUAGE.md` 了解多语言配置
